content: "<p><strong>Vercel Inc.</strong>, formerly <strong>Zeit</strong>,<sup id=\"cite_ref-BIFundingA_1-0\" class=\"reference\"><a href=\"https://en.wikipedia.org/wiki/Vercel#cite_note-BIFundingA-1\">[1]</a></sup> is an American cloud <a title=\"Platform as a service\" href=\"https://en.wikipedia.org/wiki/Platform_as_a_service\">platform as a service</a> company. The company maintains the <a title=\"Next.js\" href=\"https://en.wikipedia.org/wiki/Next.js\">Next.js</a> web development framework.<sup id=\"cite_ref-TechCrunchNextjs12_2-0\" class=\"reference\"><a href=\"https://en.wikipedia.org/wiki/Vercel#cite_note-TechCrunchNextjs12-2\">[2]</a></sup></p>\r\n<p>Vercel's architecture is built around <a title=\"Jamstack\" href=\"https://en.wikipedia.org/wiki/Jamstack\">Jamstack</a>, and deployments are handled through <a title=\"Git\" href=\"https://en.wikipedia.org/wiki/Git\">Git</a> repositories.<sup id=\"cite_ref-JamstackBook_3-0\" class=\"reference\"><a href=\"https://en.wikipedia.org/wiki/Vercel#cite_note-JamstackBook-3\">[b 1]</a></sup> Vercel is a member of the <a title=\"MACH Alliance\" href=\"https://en.wikipedia.org/wiki/MACH_Alliance\">MACH Alliance</a>.</p>\r\n<h2><span id=\"History\" class=\"mw-headline\">History</span></h2>\r\n<p>Vercel was founded by Guillermo Rauch in 2015 as Zeit.<sup id=\"cite_ref-BIFundingA_1-1\" class=\"reference\"><a href=\"https://en.wikipedia.org/wiki/Vercel#cite_note-BIFundingA-1\">[1]</a></sup><sup id=\"cite_ref-InfoWorldInfo_4-0\" class=\"reference\"><a href=\"https://en.wikipedia.org/wiki/Vercel#cite_note-InfoWorldInfo-4\">[3]</a></sup> Rauch had previously created the realtime <a title=\"Event-driven architecture\" href=\"https://en.wikipedia.org/wiki/Event-driven_architecture\">event-driven</a> communication library <a title=\"Socket.IO\" href=\"https://en.wikipedia.org/wiki/Socket.IO\">Socket.IO</a>.<sup id=\"cite_ref-InfoWorldSocketIO_5-0\" class=\"reference\"><a href=\"https://en.wikipedia.org/wiki/Vercel#cite_note-InfoWorldSocketIO-5\">[4]</a></sup> Zeit was rebranded to Vercel in April 2020, although retained the company's triangular logo.<sup id=\"cite_ref-BIFundingA_1-2\" class=\"reference\"><a href=\"https://en.wikipedia.org/wiki/Vercel#cite_note-BIFundingA-1\">[1]</a></sup><sup id=\"cite_ref-TheRegisterZeit_6-0\" class=\"reference\"><a href=\"https://en.wikipedia.org/wiki/Vercel#cite_note-TheRegisterZeit-6\">[5]</a></sup></p>\r\n<p>In June 2021, Vercel raised $102 million in a Series C funding round.<sup id=\"cite_ref-TechCrunchFundingC_7-0\" class=\"reference\"><a href=\"https://en.wikipedia.org/wiki/Vercel#cite_note-TechCrunchFundingC-7\">[6]</a></sup> As of November 2021, the company is valued at $2.5 billion.<sup id=\"cite_ref-ReutersFunding_8-0\" class=\"reference\"><a href=\"https://en.wikipedia.org/wiki/Vercel#cite_note-ReutersFunding-8\">[7]</a></sup></p>\r\n<h3><span id=\"Acquisitions\" class=\"mw-headline\">Acquisitions</span></h3>\r\n<p>On December 9, 2021, Vercel acquired Turborepo.<sup id=\"cite_ref-TechCrunchTurborepo_9-0\" class=\"reference\"><a href=\"https://en.wikipedia.org/wiki/Vercel#cite_note-TechCrunchTurborepo-9\">[8]</a></sup></p>\r\n<p>On October 25, 2022, Vercel acquired Splitbee.<sup id=\"cite_ref-SDTimesSplitbee_10-0\" class=\"reference\"><a href=\"https://en.wikipedia.org/wiki/Vercel#cite_note-SDTimesSplitbee-10\">[9]</a></sup></p>\r\n<h2><span id=\"Architecture\" class=\"mw-headline\">Architecture</span></h2>\r\n<p>Deployments through Vercel are handled through <a title=\"Git\" href=\"https://en.wikipedia.org/wiki/Git\">Git</a> repositories, with support for <a title=\"GitHub\" href=\"https://en.wikipedia.org/wiki/GitHub\">GitHub</a>, <a title=\"GitLab\" href=\"https://en.wikipedia.org/wiki/GitLab\">GitLab</a>, and <a title=\"Bitbucket\" href=\"https://en.wikipedia.org/wiki/Bitbucket\">Bitbucket</a> repositories.<sup id=\"cite_ref-GatsbyGuide_11-0\" class=\"reference\"><a href=\"https://en.wikipedia.org/wiki/Vercel#cite_note-GatsbyGuide-11\">[b 2]</a></sup> Deployments are automatically given a subdomain under the <code>vercel.app</code> domain,<sup id=\"cite_ref-InfoWorldServerless_12-0\" class=\"reference\"><a href=\"https://en.wikipedia.org/wiki/Vercel#cite_note-InfoWorldServerless-12\">[10]</a></sup> although Vercel offers support for custom domains for deployments.<sup id=\"cite_ref-GatsbyGuide_11-1\" class=\"reference\"><a href=\"https://en.wikipedia.org/wiki/Vercel#cite_note-GatsbyGuide-11\">[b 2]</a></sup></p>\r\n<p>Vercel's infrastructure uses <a title=\"Amazon Web Services\" href=\"https://en.wikipedia.org/wiki/Amazon_Web_Services\">Amazon Web Services</a> and <a title=\"Cloudflare\" href=\"https://en.wikipedia.org/wiki/Cloudflare\">Cloudflare</a>.<sup id=\"cite_ref-VentureBeatMiddleware_13-0\" class=\"reference\"><a href=\"https://en.wikipedia.org/wiki/Vercel#cite_note-VentureBeatMiddleware-13\">[11]</a></sup></p>\r\n<h2><span id=\"Reception\" class=\"mw-headline\">Reception</span></h2>\r\n<p>Vercel's clientele
